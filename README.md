# FoelliXJFX

## Examples
- Take a look at package `de.foellix.jfx.graphs.tests`, or
- Execute `java -jar FoelliXJFX-1.0.0.jar`.

## Maven
```
<dependency>
    <groupId>de.foellix</groupId>
    <artifactId>FoelliXJFX</artifactId>
    <version>1.0.0</version>
</dependency>
```
